# Ruby Learning

Weekly Ruby exercises covering fundamentals through OOP and metaprogramming.

## Structure

```
1_week/   – Basics: operators, sprintf, methods, conversions
2_week/   – Strings, conditionals, leap year
3_week/   – Arrays, hashes, iteration
4_week/   – Classes and OOP
5_week/   – Modules, inheritance
6_week/   – File I/O, exceptions
7_week/   – Metaprogramming
challenge/        – Standalone challenges
pair_programming/ – Pair programming exercises
```

Files use `e` (exercise) and `c` (challenge) prefixes per week.

## Running

```bash
ruby 1_week/6e_convert.rb
```

## Tests

Tests use Ruby's built-in [Minitest](https://github.com/minitest/minitest) framework.

```bash
# Run all tests
ruby test/run_all.rb

# Run a specific test file
ruby test/test_convert.rb
```

## Requirements

- Ruby 3.0+
